# eoffice-markdown
eoffice的markdown文件渲染器，结合eoffice-web使用

## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### 项目启动

```bash
npm start
```

### 项目构建

```bash
npm run build
```

### 代码格式检查

```bash
npm run lint
```

不合规代码快速修正:

```bash
npm run lint:fix
```

# 部署
修改`config/defaultSettings.ts`中的`basepath`为自己部署的上下文
